# Single-page-resume-website


This is a single-page resume website created using **HTML and little bit of CSS**.  
The project is designed to showcase personal details, skills, education, and contact information in a clean and simple layout.

## Features
- Single-page layout
- Clean and professional design
- Easy to customize content

## Technologies Used
- HTML5
- CSS3
